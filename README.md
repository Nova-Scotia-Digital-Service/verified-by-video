# verified-by-video

Verify users liveliness and presence by choreographed video

## Setup

Make a copy of `env.template` named `.env` in the project root.

```
cp env.template .env
```

## Development

Install:

```
yarn
```

Run development server:

```
yarn start
```

## Docker

To build and serve the app with Docker:

```
docker compose up --build
```
